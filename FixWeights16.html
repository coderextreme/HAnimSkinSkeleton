X3D Rigging Fixer: Air-Gap Culler v16

Select your X3D file. This script measures the distance a weight ray travels through "empty air" between body parts to accurately sever limbs from pulling each other.

If a ray exits a mesh and travels through open air for a distance greater than this value before hitting the next mesh (or the target), the weight is culled. This allows layered clothing (small gap) while perfectly severing arm-to-leg bleeding (massive gap). Default: 0.08 (8cm)
